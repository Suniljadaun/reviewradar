# ReviewRadar - see your feedback clearly 

A feedback intelligence Platform -- built **live on Youtube**

## what it does
ReviewRadar ingests customer reviews, classifies them with a machine-learning model ,
detects when that model degrades(drift) , retrains it automatically , and includes an AI 
assistant you can ask quesitons about the feedback -- all running on real production infrastructure.
